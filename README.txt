A website for a hotel created in PHP.

Search.php provides a search bar.
Book.php implements the booking for hotel rooms.
Pricing.php stores the price for different hotel rooms.
Register.php allows a user to register.
Showtable.php displays information stored on the website to the administrator.
Customerlogin.php controls the logging in of customers.
Logout.php is for users to logout.
Index.php is a simple index page.
Homepagecss.css is the css (design) for the homepage.
Generate-nav.php implements a dynamic navigation bar.
Adminlogin.php allows an admin to login.
